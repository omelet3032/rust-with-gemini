### 2단계: 문제 (소유권, if let, Option, Vec)

`get_inventory()` 함수가 반환하는 **`Vec<Product>`**를 인수로 받아, 재고가 **10개 미만**인 상품의 **이름 목록**을 담은 **`Vec<String>`**을 반환하는 **`get_low_stock_names` 함수**를 작성하세요.

**요구 사항:**

1. **반환 타입**: 함수는 재고가 부족한 상품의 이름 목록인 **`Vec<String>`*을 반환해야 합니다.
2. **순회 및 소유권**: `Vec<Product>`를 순회할 때, **데이터를 소비하지 않고** (`for item in vec` 형태는 피하고), **참조만 사용하여** 순회해야 합니다.
    - 힌트: `for item in &vec` 형태를 사용해야 합니다.
3. **처리 조건**:
    - *`stock` 필드가 `None`*인 경우: 재고 정보를 알 수 없으므로 무시하고 건너뜁니다.
    - *`stock` 필드가 `Some(n)`*인 경우: 내부 값 `n`이 **10보다 작은 경우** ($n < 10$)에만 해당 상품의 이름을 결과 목록에 추가합니다.
4. **코드 작성 요구사항**: `Option`의 값을 안전하게 처리하기 위해 **`if let`** 또는 **`match`** 표현식을 반드시 사용해야 합니다.
