get_user_data() 함수가 반환하는 **Vec<User>**를 인수로 받아, 다음 두 가지 조건을 모두 만족하는 사용자들의 이름 목록(Vec<String>)을 반환하는 get_active_adult_names 함수를 작성하세요.

순회 및 소유권: Vec<User>를 인수로 받은 후, 내부 데이터를 소비하지 않고 참조만 사용하여 순회해야 합니다. (함수가 종료된 후에도 Vec<User>의 원본 데이터가 유효하도록 합니다.)

클로저 및 체이닝: Vec의 메소드 체이닝 (예: .iter(), .filter(), .map(), .collect())을 사용하여 간결하게 구현해야 합니다.

필터 조건:

활성 상태: is_active 필드가 **true**여야 합니다.

성인 기준: age 필드가 20세 이상(age >= 20)이어야 합니다.

변환: 최종적으로 선택된 사용자의 username (String 타입)만 결과 목록에 포함되어야 합니다. 소유권/복사 문제를 반드시 처리해야 합니다.
